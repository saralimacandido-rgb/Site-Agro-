Sobre o site: 1. O que é o projeto

O projeto Agro Brasileiro: Soja e Sustentabilidade é um site informativo desenvolvido para o Concurso Agrinho Paraná 2026. Seu objetivo é apresentar a importância da soja para o Brasil e mostrar como a produção agrícola pode acontecer de forma sustentável, respeitando o meio ambiente.

1. O site reúne informações sobre a cultura da soja, sua contribuição para a economia brasileira e algumas práticas que ajudam a preservar os recursos naturais. Além disso, possui um quiz interativo que permite ao usuário testar seus conhecimentos sobre sustentabilidade no agronegócio.

2. Por que o projeto é necessário

A soja é um dos produtos mais importantes do agronegócio brasileiro, estando presente em diversos alimentos e sendo uma das principais responsáveis pelas exportações do país. Apesar disso, muitas pessoas ainda não conhecem sua importância e nem as ações que podem tornar sua produção mais sustentável.

Por meio deste projeto, busca-se informar e conscientizar os visitantes sobre a necessidade de equilibrar produção e preservação ambiental. Com informações simples e acessíveis, o site mostra que o desenvolvimento do agro e o cuidado com a natureza podem caminhar juntos.

3. Metodologia (Como foi desenvolvido)

O projeto foi desenvolvido seguindo as orientações do Concurso Agrinho Paraná 2026 e utilizando apenas HTML, CSS e JavaScript.

Inicialmente, foi realizada uma pesquisa sobre a soja, sua importância econômica e as principais práticas sustentáveis utilizadas no campo. Em seguida, foi criada a estrutura do site utilizando HTML5, organizando os conteúdos em seções informativas.

A parte visual foi desenvolvida com CSS3, utilizando cores inspiradas na natureza e no ambiente rural, buscando um visual agradável e de fácil navegação. Já com JavaScript, foi criado um quiz interativo que permite ao usuário responder perguntas e receber um resultado de acordo com seu desempenho.

Por fim, os arquivos serão organizados em um repositório público no GitHub e o site será publicado utilizando o GitHub Pages, conforme as regras do projeto.
